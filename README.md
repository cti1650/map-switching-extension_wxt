# Map Switching Extension

Googleストリートビューで見ている場所を他のマップで開いて表示したい！という個人的な要望をChrome拡張機能化しました！

https://chromewebstore.google.com/detail/map-switching-extension/dkpkkcfenknkfjjpbaafahfakkdlnjcl?authuser=0&hl=ja

## Command

### init

```
yarn install
```

### develop

```
yarn dev
```

### build

```
yarn build
```

### zip

```
yarn zip
```
